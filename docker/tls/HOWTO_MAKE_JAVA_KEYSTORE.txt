- Make the PKCS#12 keystore using cert + privatekey

❯ openssl pkcs12 -export -in /etc/pki/tls/certs/certificate.txt -inkey /etc/pki/tls/private/privatekey.txt -out teamcity.p12 -name teamcity -CAfile /etc/pki/tls/certs/ca-bundle.trust.crt -caname root -passout pass:Ch4n63M3

- Convert PKCS#12 to Java keystore

❯ keytool -importkeystore -deststorepass Ch4n63M3 -destkeypass Ch4n63M3 -destkeystore teamcity.jks -srckeystore teamcity.p12 -srcstoretype PKCS12 -srcstorepass Ch4n63M3 -alias teamcity
